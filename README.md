# Svenskeknappen 🇸🇪 ☎️
The button you click **before** you call customer support.

## What does it do?
It restarts the system.

## How do I use it?
1. Clone the project
2. Run `npm i`
3. Then run `npm run dev`
4. Open the website in your favorite browser
5. Click the Swedish button to restart the system 🧘🏼‍♀️
6. Done! ✅
